UMD Football Food Vendor Database

Project Description
This program manages all the food vendors and food sales at UMD football games. It allows the user to input vendors, food items, track purchases, rate the vendors, and vew the data on sales. 

Features of the Program
- Adds vendors to where they are located
- Adds food items sold by vendors
- Tracks purchases made during UMD Football games
- Rates the vendors and calculates average reating they received 
- Shows total sales for a specific football game
- Shows the best selling food item 

How to Run the Script
1. Open the program file in Python
2. Run the main Porgram in the terminal: python FinalProjectINST326.py
3. Use the menu that will appear to toggle around with the program features
